Colors and Animation

The assignment

In this assignment our purpose was to add a color pallet to a mock-up landing page and add animations and a dark theme color pallet to the site as well. I used huemint.com which generate color pallets to find the right color pallet for my site.

I used Adobes contrast analyzer to validate if the contrasts were high enough on the colors I chose. To make the dark theme I used Phina Kersly tutorial – https://www.section.io/engineering-education/adding-dark-theme-to-your-site/.

Visuel details

Buttons and animation

In this assignment I was required to add an animation to an element with JavaScript, instead of using only CSS variables. I chose to add a simple scale animation to the call to action button in the header when the user clicks on the button so the click becomes more visible.

I kept the design for the buttons on the site uniform in shape and size. The background color varies whether dark- or light mode is activated on the website. I applied the same font-family on the buttons as I used on the paragraphs to tie it together..

Dark mode

In this assignment I had to make a light and dark mode for a mock-up landing page. I started by making the light mode. I used huemint.com to find my color pallet. I then made variable in CSS for the different elements and assigned them a color. In the html file I added a data attribute called ‘data-theme=“light”’, so the default mode would be the light one.

In the CSS file I then made another CSS variable called html[data-theme=”dark”]. Here I added the same elements, but assigned them different colors, but still within the color pallet I used for the light mode. Lastly I used JavaScript to make the input checkbox in the footer toggle between the two modes.

Responsive layout

The assignment

In this assignment our purpose was to apply CSS Flex and Grid methods to a mock-up landing page, to make the layout responsive. I had to add background-images in CSS to different div’s and then place them according to the instructions for the assignment.

I used media queries with different sizes to make the site responsive. We were also encouraged to add fonts and font hierarchy, I therefor used a clamp in CSS to make the fonts responsive as well. Lastly, I used huemint.com to generate a color pallet for the website, and then applied the pallet on the elements.

Visuel details

section layout

To make the site responsive I started with designing the layout for the mobile version first. I then used various sizes in media queries to layout the content in the desktop version – depending on the section and its content.

I often used CSS Flex methods to layout the content. In the mobile version I mostly used flex-direction: column while in the desktop version I more often used flex-direction: row. To fit the headlines and paragraph into different layouts I added different max-width to them.

Fonts

I used googlefonts.com to find the fonts I wanted to use on the website, and type-scale.com to find my initial font hierarchy. Besides adding a max-width to the different text elements, I also used a clamp to make the fonts responsive.

The display font had to stand out a bit more to grab attention, so I chose a serif font for the display called ‘PT serif’. For the paragraph and UI-elements I chose a more subtle font called ‘Raleway’. Then add some extra font-weight to some of the paragraphs, where I wanted it to grab a little more attention.

Responsive layout

The assignment

In this assignment our purpose was to apply CSS Flex and Grid methods to a mock-up landing page, to make the layout responsive. I had to add background-images in CSS to different div’s and then place them according to the instructions for the assignment.

I used media queries with different sizes to make the site responsive. We were also encouraged to add fonts and font hierarchy, I therefor used a clamp in CSS to make the fonts responsive as well. Lastly, I used huemint.com to generate a color pallet for the website, and then applied the pallet on the elements.

Visuel details

section layout

In various section I used CSS Flex to layout the content. In the mobile version I mostly used flex-direction: column; while in desktop I more often used flex-direction: row;. In the ‘request’ and footer ‘section’ I also used justify-content: space-around; to place the content.

While in other sections I used space-evenly to layout the content depending on content in each section. I used various sizes in media queries to layout the content in the desktop version, depending on the section and it’s content. To fit the headlines and paragraph into different layouts I added different max-width to them.

Fonts

Besides adding max-width to the different text elements, I also used a clamp (min, val, max) to make the fonts responsive. I used type-scale.com to find my initial font hierarchy. A clamp can also be used on image tags, but since I was required to add all images as background-images I didn’t need to use it for that this time.

I used googlefonts.com to find which fonts I wanted on the website. The display font had to stand out a bit more to grab attention so I chose a serif font for the display called ‘PT serif’. For the paragraph and UI-elements I chose a more subtle font called ‘Raleway’. Then add some extra font-weight to some of the paragraphs, where I wanted it to grab a little more attention.
